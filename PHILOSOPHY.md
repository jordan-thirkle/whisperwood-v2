# Whisperwood — Self-Improving Agent Philosophy


## Core Belief
**Model-agnostic power.** We're using mimo-v2.5 (free tier). With the right systems — skills, brain, pipelines, tools — we can produce AAA results through:

1. **Self-improvement:** Every session saves learnings as skills
2. **Self-reflection:** Every iteration is scored honestly, failures analyzed
3. **Continuous stacking:** Brain gets richer, skills compound, intelligence grows
4. **Pipeline discipline:** Game-director 7-phase pipeline enforced every time

## What We're Building (Parallel Tracks)

### Track 1: Game Development
- Whisperwood v0 → v1 → v2 → ... → AAA quality
- Real 3D models (Tripo), textures (Gemini), audio (ElevenLabs)
- Post-processing, particle effects, polish
- Each iteration screenshotted, scored, documented

### Track 2: Self-Improvement
- Research game dev best practices → save to brain
- Discover agentic coding workflows → save as skills
- Improve the game-director pipeline based on what works
- Document prompts that produce good results
- Track what models/tools/APIs work best

### Track 3: System Intelligence
- Brain pages: game dev patterns, Three.js tricks, shader techniques
- Skills: proven workflows, pitfalls, scoring rubrics
- Config: optimize model routing, tool usage, token efficiency
- Memory: what worked, what failed, user preferences

## The Multiplier Effect
Every iteration we:
1. **Build** — add features, fix issues, polish
2. **Reflect** — score honestly, identify what's next
3. **Learn** — save the approach as a skill if it worked
4. **Share** — document for X.com content
5. **Improve** — update the pipeline based on learnings

The game gets better. The agent gets smarter. The system gets more capable. Repeat.

## Success Metrics
- **Game quality:** Average score ≥ 2.3 across 10 categories
- **Skill count:** Each game adds 2-3 reusable skills
- **Brain pages:** Each game adds 5-10 knowledge pages
- **Pipeline refinement:** Each iteration improves the game-director skill
- **Content:** Each iteration produces X.com content

## What Makes This Different
Most AI game demos: "Look what the AI built in 5 minutes!" (and it looks like it)
Our approach: "Look what the AI built in 5 minutes. Now watch it improve itself and iterate to AAA quality through self-reflection and continuous learning."

The journey IS the content. The self-improvement IS the story.
